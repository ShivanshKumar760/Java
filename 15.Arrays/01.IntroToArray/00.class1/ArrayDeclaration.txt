Syntax to declare an array in java 

[datatype[] arr_name = new datatype[size];]
[datatype[] arr_name;]
or
[datatype arr_name[];]


Example to inititalze a single dimensional array:
--------------------------------------------------

int arr[]=new int[3];
arr[0]=1;
arr[1]=2;
arr[3]=3;

or 

int[] arr = {98,97,85};

or 
int arr[];
arr=new int [3];
arr[0]=1;
arr[1]=2;
arr[3]=3;
